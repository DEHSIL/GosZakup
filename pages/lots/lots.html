Уважаемые потенциальные поставщики!

В целях исключения случаев, ранее возникавших на веб-портале государственных закупок при шифровании ценовых предложений, таких как дублирование ценовых предложений, некорректное отображение ценовых предложений, указание цен ниже допустимого уровня демпинга, необходимо обновить криптосокет до последней версии.
Просим вас установить актуальную версию криптосокета, который можно скачать по ссылке для ОС Windows (ссылка), для ОС Mac (ссылка).
Использование устаревших версий может привести к некорректной подаче ценовых предложений и невозможности их дальнейшей расшифровки.
В связи с чем с 5 сентября 2025 года система будет блокировать подачу ценовых предложений в случае необновленной версии криптосокета.

На ОС Mac рекомендуется обновить ПО для стабильной работы CryptoSocket.
В случае если Mac на Apple Silicon, может потребоваться установка Rosetta - также macOS предлагает это автоматически при первом запуске установки CryptoSocket.
Отмечаем, что вышеуказанные версии криптосокета не подходят для использования в банковских организациях.
Для определения подходящего криптосокета следует обращаться в тот банк, с которым вы сотрудничаете. Также ознакомиться с инструкцией по установке: для ПО на ОС Windows (ссылка), для ПО на ОС Mac (ссылка).
